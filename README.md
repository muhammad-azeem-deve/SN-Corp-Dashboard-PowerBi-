# SN-Corp Dashboard (Power BI)

## 📊 Project Overview

The **SN-Corp Dashboard** is an interactive data visualization project built using **Power BI**. It provides a clear overview of sales performance, costs, profits, customer activity, and regional distribution. The dashboard helps users analyze business data quickly and make informed decisions through visual insights.

## 🎯 Objectives

* Monitor overall **sales, cost, and profit performance**
* Analyze **customer sales contribution**
* Visualize **product sub-category performance**
* Track **order trends by year and quarter**
* Understand **regional sales distribution**

## 🛠 Tools & Technologies

* **Power BI**
* Data Visualization
* Business Intelligence
* Interactive Filters & Slicers

## 📈 Dashboard Features

* **Total Sales, Total Cost, and Total Profit KPIs**
* **Customer-wise sales and profit analysis**
* **Sub-category performance bar chart**
* **Interactive map showing sales by region**
* **Order trends by year and quarter**
* **Total quantity distribution by region**
* **Filters for Order Date and Ship Mode**

## 🔎 Key Insights

* Quickly identify top-performing **customers and product categories**
* Compare **sales across different regions**
* Track **business growth trends over time**
* Analyze **shipping modes and their impact**

1. Download the `.pbix` file from this repository.
2. Open the file using **Microsoft Power BI Desktop**.
3. Interact with the filters and visuals to explore the data.
